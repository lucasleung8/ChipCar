# ChipCar-IEEE

A full-stack remote control car controllable anywhere in the world, built by the Raspberry Pi team at McMaster. We used a Raspberry Pi alongside Python, Flask, JavaScript, WebSockets (SocketIO) for real-time communication and user authentication, live video stream using OpenCV, threaded concurrency with gunicorn, and securely deployed via Cloudflare Tunnel. Try it at [chipcar.cc](https://chipcar.cc)!

## How To Run
Run run.sh in your terminal, changing the motor pin numbers if required.
